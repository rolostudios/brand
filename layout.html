Layout

Small screens

Use a flexible one column layout with 24px gutters on the side on iOS, Android and mobile web.

Medium and large screens

Use a 12 column grid with 24px inner gutters and 24px outer gutter. Medium screens use a fixed width centered 12 column layout (min-width: 744px, 696px content width with 24px gutters on the side). Large screens use a fixed width centered 12 column layout (min-width: 1128px, 1080px content width with 24px gutters on the side)

All screens

Use an 8px baseline grid for vertical and horizontal components and spacing when it makes sense. We use a 4px baseline grid for typography. This makes the type leadings work well with components and spacing. Exceptions can be made for native interface elements like iOS Nav and Tabbars.
